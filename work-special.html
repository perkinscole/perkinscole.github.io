Remy the Robot

Jetson Orin Nano · Autonomous Robotics In Progress

Remy is a classroom robot built on an NVIDIA Jetson Orin Nano, a platform powerful enough to run real computer-vision and ML workloads on-device. The plan is for Remy to become a shared classroom character students can interact with, program, and extend: a physical anchor for AI and robotics units that gives abstract concepts a face (and wheels).
