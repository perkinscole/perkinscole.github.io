# Cole Perkins — Personal Website

A multi-page static website for Cole Perkins, middle school computer science teacher at Robert Adams Middle School in Holliston, MA, with a background in software development.

## About

This site showcases projects, curriculum, and tools I've built at the intersection of education, design, and technology. It includes work samples across student-facing projects, curriculum design, web apps, special projects, and clubs.

## Site Structure

- **Home** — Introduction and current work
- **About** — Background, teaching philosophy, and interests
- **Work** — Portfolio organized by category:
  - Student-Facing (Rams Care, Guess That Teacher)
  - Curriculum (VEX robotics, Creative Computing, AI & Ethics)
  - Apps (PBIS dashboard, Unified Arts site, RAMS Heartbeat, Curriculum platform)
  - Special Projects (micro:bit drones, multiplayer Scratch, Remy)
  - Clubs (Computer Club, Camera Club)
- **CV** — Full curriculum vitae
- **Contact** — How to get in touch

## Built With

Plain HTML, CSS, and a love of whitespace. No frameworks, no build step. Fonts are Playfair Display and Source Sans 3.
